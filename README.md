# Exprtmpl
一个C#实现的文本模板引擎

模板引擎是很古老的技术了，除了用于web渲染外，代码生成也可以使用。因为我没有找到特别符合我需求的C#实现的模板引擎(这只是一个造轮子的理由)，因此自己写了一个。本项目的特点如下：
1. 以类Json数据结构为驱动渲染，类型严格(可映射任意类型)
2. 支持简单的逻辑运算
3. 支持引入其他模板
4. 支持添加自定义函数
